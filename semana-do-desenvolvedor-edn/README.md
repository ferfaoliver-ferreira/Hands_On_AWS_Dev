# Semana do Desenvolvedor EDN

Esta pasta centraliza os laboratorios praticos desenvolvidos durante a **Semana do Desenvolvedor da Escola da Nuvem**. A proposta e documentar os quatro dias em uma unica trilha dentro do repositorio principal [Hands_On_AWS_Dev](../).

## Objetivo desta trilha

Organizar os laboratorios da semana em um formato consistente, deixando cada dia com:

- documentacao tecnica
- contexto do laboratorio
- servicos AWS utilizados
- passo a passo
- evidencias visuais
- aprendizados consolidados

## Estrutura dos 4 dias

| Dia | Pasta | Status | Descricao |
| :--- | :--- | :--- | :--- |
| Dia 1 | [dia-1-api-eventbridge](./dia-1-api-eventbridge) | Documentado | Ingestao de pedidos via API Gateway, Lambda, SQS FIFO, DLQ e EventBridge. |
| Dia 2 | [dia-2](./dia-2) | Em preparacao | Espaco reservado para o segundo laboratorio da semana. |
| Dia 3 | [dia-3](./dia-3) | Em preparacao | Espaco reservado para o terceiro laboratorio da semana. |
| Dia 4 | [dia-4](./dia-4) | Em preparacao | Espaco reservado para o quarto laboratorio da semana. |

## O que voce vai encontrar aqui

Ao longo da trilha, a ideia e mostrar a evolucao dos estudos praticos de Developer na AWS, consolidando varios padroes importantes, como:

- ingestao de eventos
- processamento assicrono
- desacoplamento com filas
- publicacao em barramentos de eventos
- observabilidade
- seguranca com IAM

## Observacao

O **Dia 1** ja esta documentado com estrutura completa. Os **Dias 2, 3 e 4** permanecem como placeholders para receber os proximos materiais no mesmo padrao.
