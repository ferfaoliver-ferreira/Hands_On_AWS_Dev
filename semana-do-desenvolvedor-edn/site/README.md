# Frontend da Semana do Desenvolvedor EDN

Esta pasta contém uma interface web estática para apresentar a trilha da **Semana do Desenvolvedor EDN** de forma mais visual.

## Arquivos principais

- `index.html`
  página inicial da trilha
- `dia-1.html` até `dia-4.html`
  páginas individuais de cada etapa
- `styles.css`
  identidade visual compartilhada
- `app.js`
  conteúdo estruturado e renderização das páginas

## Como abrir

Você pode abrir `index.html` diretamente no navegador ou servir a pasta com um servidor estático simples.

Exemplo com Python:

```bash
python -m http.server 4173
```

Depois disso:

- acesse `http://127.0.0.1:4173/index.html`
